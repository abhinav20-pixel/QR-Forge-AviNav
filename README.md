# ✨ Overview

**QR Forge - AviNav** is a modern web application that allows users to generate multiple types of QR codes instantly.

The application is designed with a **premium Claymorphism UI**, smooth animations, and a modern dashboard experience. It simplifies sharing payment links, WiFi credentials, WhatsApp contacts, and universal content using QR codes.

This project is a **frontend-focused web tool built using HTML, CSS, and JavaScript**.

---

# 🎯 Features

### 🏠 Animated Dashboard

Modern homepage with animated UI components and quick access tools.

### 💳 UPI QR Generator

Generate QR codes for **UPI payments** compatible with all major UPI apps.

### 💬 WhatsApp QR Generator

Create QR codes that instantly open WhatsApp chats with a specific number.

### 📶 WiFi QR Generator

Generate QR codes to easily share WiFi network credentials.

### 🔗 Universal QR Generator

Generate QR codes from:

* Text
* URLs
* Images
* Custom input

### 🎨 Premium UI

* Claymorphism design
* Soft shadows
* Modern gradients
* Smooth transitions
